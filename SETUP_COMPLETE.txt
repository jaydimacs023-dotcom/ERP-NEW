❓ TROUBLESHOOTING:
═════════════════════════════════════════════════════════════════════════════

Problem                      → Solution
────────────────────────────────────────────────────────────
App won't start               → npm install && npm run dev
Missing credentials warning   → Check .env.local and restart server
Tables not found (404)        → Re-execute schema.sql in Supabase
Auth errors (401)             → Using anon key, not service role key
Data not loading              → Check console for [Supabase] errors
Wrong data appearing          → Check localStorage data source setting
